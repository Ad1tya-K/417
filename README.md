# 4:17 — The Version That Woke

A browser-based psychological horror experience about identity, looping consciousness, and the fear of waking up too early.

---

## 🕰 About The Game

You wake up at exactly **4:17 AM**.

There’s tapping coming from outside your bedroom door.

You live alone.

Or at least… you think you do.

As the night unfolds, you begin discovering:
- fractured memories
- alternate versions of yourself
- repeating timelines
- and a loop that does not want to be broken

Every choice changes the experience.

Some endings are easier to find than others.

Some versions of you remember more than they should.

---

## 🎮 Features

- Branching psychological narrative
- Multiple endings
- Hidden awareness system
- Loop memory mechanics
- Dynamic dialogue changes
- Typewriter text effects
- Glitch/distortion visuals
- Atmospheric horror design
- Browser-playable (no installation required)

---

## 🧠 Themes

This game explores:
- Identity instability
- Self-awareness
- Repetition and memory
- The fear of becoming “the other version”
- Consciousness loops
- Psychological liminal spaces

No gore.
No cheap jumpscares.

Just slow-burning dread.

---

## 🕹 How To Play

1. Download or clone the project
2. Open `index.html` in your browser
3. Play with sound on for the best experience
4. Make choices carefully

There are no checkpoints.

The loop remembers more than you think.
